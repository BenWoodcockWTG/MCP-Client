# MCPClient (.NET 10 Console App)

Skeleton for a .NET 10 command-line application.

## Project structure

- `MCPClient.slnx` — solution file
- `src/MCPClient/MCPClient.csproj` — console app targeting `net10.0`
- `src/MCPClient/Program.cs` — app entry point

## Build

```bash
dotnet build MCPClient.slnx
```

## Run

```bash
dotnet run --project src/MCPClient
```
